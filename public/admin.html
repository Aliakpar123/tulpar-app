Telegram интеграция

Настройки уведомлений в Telegram

Как настроить уведомления:
1️⃣ Создайте бота через @BotFather в Telegram
2️⃣ Получите токен вида 123456:ABC...
3️⃣ Узнайте свой Chat ID через @userinfobot
4️⃣ Вставьте значения в файл .env
5️⃣ Перезапустите сервер
TELEGRAM_BOT_TOKEN=1234567890:AAxxxxxxxxxx
TELEGRAM_ADMIN_CHAT_ID=987654321
После настройки каждый новый партнёр будет отправлять уведомление с именем, телефоном, городом и выбранным тарифом прямо в ваш Telegram.
